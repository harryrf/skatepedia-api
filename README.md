# Skatepedia

A REST API for skate scraped data.


## Installation

This project runs an API with acces to the scrapped data. (Django + rest framework + Scrapy)

`pipenv install`


Run in local:


`pipenv shell python manage.py runserver`



## Crawl data

`scrapy crawl <spider_name>`
